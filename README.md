# زیرنویس — مترجم فایل SRT

وب‌اپلیکیشنی برای ترجمهٔ فایل‌های زیرنویس `.srt` با استفاده از Claude. شماره‌ها و تایم‌کدهای فایل دست‌نخورده باقی می‌مانند و فقط متنِ هر پاراگراف ترجمه می‌شود.

## امکانات
- بارگذاری فایل `.srt` (کشیدن‌ورهاکردن یا انتخاب دستی)
- انتخاب زبان مبدأ و مقصد (دلخواه)
- سه لحن ترجمه:
  - **محاوره‌ای و کوچه‌بازاری**
  - **محاوره‌ای، باکلاس و شیک**
  - **رسمی و حقوقی**
- ترجمهٔ سلیس و گویا؛ اسم‌های خاص انگلیسی به‌صورت آوانگاری فارسی داخل گیومه (مثلاً `Claude Cowork` ← `"کلاود کوورک"`)
- نمایش پیشرفت لحظه‌ای و دانلود فایل ترجمه‌شده

## اجرای محلی
کلید API به‌صورت متغیر محیطی خوانده می‌شود (هرگز داخل کد نوشته نمی‌شود):
```powershell
# PowerShell
$env:ANTHROPIC_API_KEY = "sk-ant-..."
$env:ACCESS_CODE = "یک-کد-دلخواه"   # اختیاری؛ برای محافظت از اعتبار
node server.js
```
سپس مرورگر را روی <http://localhost:3000> باز کنید.

## متغیرهای محیطی
| متغیر | نقش | پیش‌فرض |
| --- | --- | --- |
| `ANTHROPIC_API_KEY` | کلید API (الزامی) | — |
| `ACCESS_CODE` | کد دسترسی مشترک؛ اگر تنظیم شود کاربر باید واردش کند | خالی (بدون محدودیت) |
| `TRANSLATE_MODEL` | مدل ترجمه | `claude-sonnet-5` |
| `MAX_CUES` | سقف تعداد پاراگراف در هر فایل | `3000` |
| `PORT` | پورت سرور | `3000` |

## انتشار عمومی روی Render (رایگان)
1. کد را روی یک ریپازیتوری GitHub بگذارید (فایل `.gitignore` جلوی کامیت‌شدن کلید/فایل‌های تست را می‌گیرد).
2. در <https://render.com> وارد شوید → **New → Web Service** و ریپو را وصل کنید (یا از `render.yaml` با گزینهٔ Blueprint استفاده کنید).
3. تنظیمات: Build Command = `npm install`، Start Command = `node server.js`.
4. در تب **Environment** این مقادیر را اضافه کنید:
   - `ANTHROPIC_API_KEY` = کلید شما (به‌صورت Secret)
   - `ACCESS_CODE` = یک کد دلخواه که فقط به افراد موردنظرتان می‌دهید
5. Deploy کنید؛ یک لینک عمومی مثل `https://zirnevis.onrender.com` می‌گیرید.

> ⚠️ کلید API فقط در تنظیمات Render ذخیره می‌شود، نه در کد. هرگز آن را داخل فایل‌ها ننویسید و کامیت نکنید.
> نکته: پلن رایگان Render پس از چند دقیقه بی‌کاری می‌خوابد و درخواست بعدی ۳۰ تا ۶۰ ثانیه طول می‌کشد تا بیدار شود.

## ساختار
| فایل | نقش |
| --- | --- |
| `server.js` | سرور HTTP بدون وابستگی + استریم پیشرفت (SSE) |
| `srt.js` | تجزیه و بازسازی فایل SRT (حفظ شماره و تایم‌کد) |
| `translate.js` | موتور ترجمه، دسته‌بندی پاراگراف‌ها و فراخوانی Claude API |
| `public/` | رابط کاربری (HTML/CSS/JS) |

هیچ وابستگی npm‌ای لازم نیست؛ فقط Node.js نسخهٔ ۱۸ به بالا.
